# Lr3_Area
_Консольная_ программа на **С#** для расчета площадей разных геометрических фигур.  
## Формулы:
параллелограмм - S = a * h | P = 2 * (a(ширина) + b(высота))  
квадрат - S = a * a | P = 4 * a  
треуголник - S = √(p*(p-a)*(p-b)*(p-c)) | P = a + b + c  
трапеция - S = (a + b) * h / 2 | P = a + b + c + d
